- Authored full **Altium Designer** schematic and **4-layer PCB layout** following industry standards
- `STM32` `IoT` `Altium Designer` `PCB Design` `Low-Power Design`

#### 🤖 Autonomous Competition Robot
Autonomous differential-drive robot with closed-loop navigation.
- Engineered **C++ firmware** and state-machine logic for robust algorithmic control

- Managed complete sensor-to-MCU integration, debugging hardware-software timing constraints with oscilloscopes and logic analyzers
- `Arduino` `C++` `OOP` `PlatformIO` `PID Control` `TOF Sensors`

- Computed real-time relative distance of rear-quarter vehicles using arrays of **ultrasonic transducers**
- Leveraged **ESP32 dual-core architecture** for parallel left/right blind-spot detection pipelines
- Authored non-blocking, multi-threaded **C++** algorithms to filter environmental vibration and acoustic noise
- `ESP32` `C++` `Ultrasonic Sensors` `ADAS` `Multi-threading`
